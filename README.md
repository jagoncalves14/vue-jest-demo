# Vue Jest Demo

Basic examples on how to write unit tests for a Vue application using Jest.

## Project setup

### Step 1. Install dependencies

```
npm run install
```

### Step 2. Compilation

#### Compiles and hot-reloads for development2

```
npm run serve
```

#### Compiles and minifies for production

```
npm run build
```
